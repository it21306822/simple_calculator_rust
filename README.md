# simple_calculator_rust


This is a basic command-line calculator written in Rust. It supports addition, subtraction, multiplication, and division operations.

## Features

- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`) with error handling for division by zero
